# 🎓 Udemy Clone

A modern and responsive **Udemy Clone** built using **HTML5** and **CSS3**. This project recreates the UI of the Udemy homepage with a clean, attractive, and mobile-friendly design.

## ✨ Features

- 🎨 Modern and Premium UI Design
- 📱 Fully Responsive Layout
- 🖼️ Hero Banner Section
- 🔍 Search Bar
- 📚 Course Categories
- ⭐ Recommended Courses
- 🔥 Most Popular Courses
- 💼 Topics Section
- 📖 Footer Section
- ⚡ Clean HTML & CSS Code
- 🌐 Cross Browser Compatible

---

## 🛠️ Built With

- HTML5
- CSS3
- Google Fonts
- Font Awesome
